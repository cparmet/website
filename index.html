PROJECTS

Top 20% finish in Kaggle competition personalized medicine



I built a model using natural language processing (NLP) to predict the effects of genetic mutations on cancer for a Memorial Sloan Kettering competition. I learned a lot about cleaning text data, using NLP and text mining, and laying pipelines for machine learning.

Tools: machine learning (XGboost, LASSO, Elastic-net, Ridge, SVD, CV), natural language processing & text mining (TF-IDF, Regex), Scipy, Python

citation formatting with NLP



I made a web app for my Healthwise team to format research citations in our house style. It was used in workflow over 100 times in 2018. It includes an algorithm to convert article titles from Title Case to Sentence case, a task that commercial reference managers punt on. (For example, try entering PMID 28859415.) I learned gobs about NLP, text mining, and deploying web apps on AWS.

Tools: NLP (NLTK, Truecasing), Text mining, Python, APIs (PubMed), Flask, Amazon Web Services (AWS S3, Lambda), Biopython

Predict traffic jams due to trains



Tired of waiting for trains to cross the road in Downtown Framingham? I made Trainingham, a Python web app, to give a heads up the next time a commuter train will stop traffic. Plan ahead and never be late for a haircut again! It taught me how to work responsibly with dates and times, including not assuming the time zone of the AWS server where an app may be running.

Tools: JSON parsing, Python (Flask, requests, datetime, pytz), AWS (S3, Lambda), APIs (MBTA), time data

PowerBI dashboards and reports



I respond to internal requests for data reports and analyses. I've created dashboards for self-service data and made a dashboard for a client (large hospital) as a demo. It's been eye-opening for me regarding the importance of understanding data provenance (I've spent more time asking people questions than working with data), how to work with different kinds of analytics data from a warehouse, and guidelines on user interface design for dashboards.

Tools: PowerBI, Microsoft SQL Server, Excel

Open source contribution to Biopython



While building the web app Bella, I proposed a change to Biopython, an open source Python package for biological computing. The change enables a piece of Biopython to work on server-less deployments, such as AWS Lambda and Google Cloud Functions. The change was integrated into Biopython. I learned about unit testing, pull requests, collaborating on GitHub, and the joys of contributing to an open source project.

Tools: Python (Biopython), AWS (Lambda), GitHub, open source projects, APIs (PubMed/Entrez)

Scrape the web for the best electricity rates



My electricity supplier lets you update your contract when the price of electricity goes down. But you need to check a website regularly to see if the rates have changed. Since we live in the future, I scheduled a script to scrape the web page daily and email me if the rate has gone down. It taught me about web scraping (I’m still tickled that you can automate filling out forms), scheduling scripts, logging, and emailing in Python.

Tools:Web scraping (Beautiful Soup, Selenium, UI automation), Python (Pandas, Flask), SendGrid.
